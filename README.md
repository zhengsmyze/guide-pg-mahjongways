# guide-pg-mahjongways

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容主要围绕特定的主题或工具，以静态 HTML 文件的形式存放，便于直接访问和浏览。

本仓库不针对任何特定网站或域名，仅作为页面存档与展示的载体。

## 目录说明

仓库根目录下存放若干 HTML 文件，每个文件为一个独立页面。目录结构如下：

```
guide-pg-mahjongways/
├── index.html          # 入口页面（如存在）
├── page1.html          # 示例页面
├── page2.html          # 示例页面
└── ...
```

页面之间无相互依赖关系，均可单独打开使用。

## 页面归档说明

- 所有页面均为纯 HTML 格式，不依赖外部数据库或后端服务。
- 页面内可能包含少量 CSS 或 JavaScript，用于实现基本交互或样式。
- 归档的页面内容保持原始状态，不进行修改或更新。
- 如有新增页面，会直接添加至根目录，并在必要时更新索引。

## 维护说明

- 本仓库由维护者不定期更新，主要工作为添加新页面或修复显示问题。
- 欢迎通过 Issue 报告问题或提出建议，但请注意不要附带具体网址或营销内容。
- 不接受涉及商业推广、诱导点击或违反平台规定的页面提交。
- 仓库不保证所有页面的长期可用性或兼容性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 HTML 文件即可查看对应页面。
3. 如需部署到静态托管服务，可将整个仓库内容上传。
